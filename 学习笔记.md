# CMD命令

1. dir：查看当前路径下的内容
2. cls：清屏
3. cd \：回到盘符的根目录
4. exit：退出窗口

# 环境变量

目的：想要在**任意目录下**都可以打开指定的软件

系统变量-Path：记录可执行文件的路径		（顺序查找）

## 配置

1. 配置JAVA_HOME系统变量：路径不带bin
2. 配置Path：%JAVA_HOME%\bin

# java代码运行步骤

1. 编写程序：文件后缀名必须是java

2. 编译文件：翻译

   ```
   javac xxx.java
   ```

3. 运行程序

   ```
   java xxx // 这里不带后缀
   ```
